pyendeavor.api.get_rides(clubId=None, athleteId=None, athleteName=None, startDate=None, endDate=None, startId=None, offset=None)[source]¶

Get a listing of the rides based on provided criteria. Rides returned will be limited to 50.

Parameters:
  • clubId – Id of the Club for which to search for member’s Rides.
  • athleteId – Id of the Athlete for which to search for Rides.
  • athleteName – Username of the Athlete for which to search for Rides.
  • startDate – Day on which to start search for Rides. YYYY-MM-DD
  • endDate – Day on which to end search for Rides.
  • startId – Return Rides with an Id greater than or equal to the startId
  • offset – Return Rides at offset
Returns:

A list of dicts that represent individual rides
